# Network Traffic Analysis using Wireshark

## 📌 Objective
To analyze network traffic and identify suspicious activities using Wireshark.

## 🛠 Tools Used
- Wireshark
- Sample PCAP files

## 🔍 Analysis
- Captured network packets (HTTP, DNS, TCP)
- Observed communication between IP addresses
- Identified unusual traffic patterns

## 🚨 Findings
- Suspicious DNS queries detected
- Unknown IP communication observed
- Possible malicious traffic pattern
